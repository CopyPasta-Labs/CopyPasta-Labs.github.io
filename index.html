Portfolio

Privacy Pools – Private ETH Transactions

Private ETH transactions powered by zero-knowledge proofs and on-chain compliance.

A full-stack implementation of the Privacy Pools protocol. Deposit ETH from one wallet and withdraw to another with no on-chain link between them — powered by zero-knowledge proofs and commitment schemes. The system integrates an Association Set Provider (ASP) for regulatory compliance, a ZK relayer that submits withdrawals on behalf of users to preserve privacy, a block explorer, and a polished Next.js web interface. The entire stack is containerized and ships as a single docker compose up.
